# Shadab Khan - Portfolio (Next.js + Tailwind)

This is a ready-to-deploy Next.js portfolio tailored for a Data Analyst / ML Developer.
It uses Tailwind CSS and is configured for easy deployment on Vercel.

## How to run locally

1. Install dependencies:
```bash
npm install
```

2. Run dev server:
```bash
npm run dev
```

3. Open http://localhost:3000

## Deploying to Vercel (recommended)
1. Push this repository to GitHub.
2. Sign in to Vercel and import the GitHub repo. Vercel auto-detects Next.js and deploys.
3. Alternatively, use `vercel` CLI to deploy from your local directory.
